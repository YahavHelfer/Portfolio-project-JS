Tic Tac Toe

This JavaScript-based Tic-Tac-Toe game offers a fun and interactive way to enjoy the classic game. Featuring a sleek and responsive design with an appealing orange and light blue color scheme, the game allows two players to compete by marking X's and O's on a 3x3 grid. The interface highlights the winning combination and provides a prompt to play again, ensuring an engaging user experience.
