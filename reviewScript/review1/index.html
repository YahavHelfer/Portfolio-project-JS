Contries API

This project utilizes a public API to provide detailed information about countries worldwide. Built with JavaScript, the application allows users to search for countries by name and view comprehensive data, including population, capital, region, and flag. The intuitive interface ensures a seamless user experience, making it easy to explore and learn about different countries.
